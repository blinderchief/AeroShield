# AeroShield

AI-Augmented Parametric Travel Insurance on Flare Network

## What is AeroShield?

AeroShield is a revolutionary decentralized flight insurance platform that:

- **Uses AI** (Google Gemini) for real-time flight delay prediction and dynamic premium pricing
- **Leverages Flare Network** for trustless flight data attestation via FDC (Flare Data Connector)
- **Provides Instant Payouts** when flight delays or cancellations are verified on-chain
- **Offers DeFi Yields** to liquidity providers who underwrite policies

## Hackathon Submission

This project was built for the Flare Network Hackathon, showcasing the unique capabilities of:

1. **Flare Data Connector (FDC)**: Trustless flight status verification
2. **FTSO v2**: Real-time price feeds for premium calculations
3. **Smart Accounts**: Gasless transactions for seamless UX

## Tech Stack

- **Frontend**: Next.js 14, React 18, Tailwind CSS, Framer Motion
- **Backend**: FastAPI, SQLAlchemy, Pydantic
- **Blockchain**: Solidity, Hardhat, ethers.js
- **AI**: Google Gemini 1.5 Flash
- **Auth**: Clerk
- **Database**: Neon PostgreSQL
